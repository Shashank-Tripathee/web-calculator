# Web Calculator

A fully functional, modern **basic arithmetic web calculator** built with plain HTML, CSS, and JavaScript — no frameworks or dependencies required.

## Features

- **Basic Arithmetic** — Addition, Subtraction, Multiplication, Division
- **Percentage** — Quick percent calculations
- **Decimal Numbers** — Full floating-point support
- **Chained Operations** — e.g. `5 + 3 × 2`
- **Division by Zero Handling** — Displays a friendly error message
- **Backspace** — Delete the last entered character
- **Clear** — Reset the calculator instantly
- **Real-time Display** — Shows input and result as you type
- **Keyboard Support** — Use your keyboard for fast input
- **Responsive Design** — Works on desktop, tablet, and mobile

## How to Use

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Start calculating!

No build step, no server, no dependencies — just open and use.

## Technologies Used

| Technology | Purpose |
|---|---|
| HTML5 | Semantic structure and accessibility (ARIA) |
| CSS3 | Styling, CSS Grid layout, animations, responsive design |
| JavaScript (ES6+) | Calculator logic, event handling, keyboard support |

## Keyboard Shortcuts

| Key | Action |
|---|---|
| `0` – `9` | Enter digits |
| `.` | Decimal point |
| `+` | Addition |
| `-` | Subtraction |
| `*` or `x` | Multiplication |
| `/` | Division |
| `%` | Percent |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last character |
| `Escape` or `Delete` | Clear / Reset |
